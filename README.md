  Trabajo final 
  Este proyecto consiste en el desarrollo del juego clasico piedra papel o tijeras programado en python, el usuario juega contra la computadora la cual genera su eleccion de manera aletoria.
  El programa permite jugar multiples rondas y valida que las entradas del usuario sean correctas.
  Objetivo Principal 
  Aplicar conceptos basicos de programacion, Implementar estructuras condicionales, Utilizar ciclos, Validar datos ingresados, Generar opciones aleatorias.
  La creacion de este software dedmuestra que un simple juego puede convertirse en un ejercico practico de logica de programacion. 
  Principales Funcionalidades 
  Permite al usuario elejir entre: piedra, papel o tijeras.
  la computadora selecciona una opcion aleatoria.
  El software: detecta empate, determina si el usuario pierde, determina si el usuario gana. 
  Valida que el usuario ingrese opciones correctas.
  Permite jugar multiples rondas.
  Solicita confirmacion para volver a jugar (solo acepta si o no) 
  Herraminetas utiilizadas: visual studio, python, libreria random
  Este proyecto me ayudo a adquirir conociminetos basicos sobre programacion, pensamiento logico y validacion de datos, demostrando que incluso un juego sencillo puede ayudar a desarrollar habilidades
  fundamentales en desarrollo de software.
